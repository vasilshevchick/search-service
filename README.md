# Simple search engine

## Run Application locally

Build project
### `mvn clean package`

Run Server
### `java -jar search-server/target/search-server-0.0.1-SNAPSHOT.jar`

Run client
### `java -jar search-client/target/search-client-0.0.1-SNAPSHOT.jar`

Search shell commands example
### `help`
### `help search`
### `search --tokens data`
